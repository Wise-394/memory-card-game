# 🎮 Dota 2 Memory Card Game

**A fan-made memory card game test your Dota 2 hero knowledge without clicking the same hero twice.**

[![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB)](https://reactjs.org/)

[Live Demo](https://github.com/wise-394)

</div>

---

## ✨ Features

- **Random Heroes** — A new set of Dota 2 heroes is fetched every game via the OpenDota API
- **Shuffle on Click** — Heroes shuffle after every click to challenge your memory
- **Score Tracking** — Tracks your current score and best score
- **Responsive Design** — Works on both desktop and mobile

## 🎯 How to Play

1. A grid of Dota 2 hero cards is displayed
2. Click on a hero you **have not clicked before**
3. The cards shuffle after every click
4. Click the **same hero twice** and you lose
5. Click all heroes without repeating to win!
